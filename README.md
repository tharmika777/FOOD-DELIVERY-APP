# 🍔 FoodieGo (Zomato/Swiggy Clone)

FoodieGo is a simple **food delivery website** inspired by **Zomato/Swiggy**, built using **HTML, CSS, and JavaScript** in a single file.

## ✨ Features
- Restaurant listing with images
- Search restaurants/food
- Filter by categories
- View menu with food images
- Add to cart & update quantity
- Total bill calculation
- Checkout alert popup

## 🛠 Tech Used
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Download the project
2. Open `index.html` in any browser

## 📌 Note
This is a frontend-only project (no backend/database).
